Project Documentation

Zicong Wei

Mission

This is a website for travelers in Beining for finding local foods

There are pages for food information, location, and suggestions. Also, it includes fun interaction

Forms

Test

I use form to make a small survey, with five questions. It can test people's taste on northern or southern Chinese food. Besides, the answer are collected for future use.

More

Use the form to subscribe

Cookies

Test Result Suggestion

Cookies are used in store the answers of the test. With the results of taste, I give food suggestions.

Favorite

Furthermore, I add a Like button, so that users can store the food they interested in to the favorite list. In this part, I use an Array and session to do this cross-page storage, instead of using cookies.

Interaction

Food Map

There are many interactions. The most important one is the food map. Users can click the markers to explore restaurant location. Besides, there are forms and rollover effect.

User Feedback

More

User can contact me, filling out the form.

Canvas Graphics

Test Result

A circle to show the loading effect, and show the percentage.

Multimedia

Home More

Background music in the home page. Users can click the red icon to stop or plat the BGM. Also, in More page, I put youtube videos about beijing street food for users to explore more information.

Project1 Element

This website uses CSS and JS. It also uses lots of images and links. The lay out is clear using div. It has a home page and navigation bar.
